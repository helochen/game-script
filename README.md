系统环境：
    win10 + python3 + conda管理 


本教程分四部分：

（1）使用python 的pywin32这个库来实现对Windows窗口的操作和点击。

（2）使用PIL来抓取图像。

（3）使用CV2实现识别图片上按钮与任务物品。

（4）easyOcr读取图片上的文字，获取到任务所需物品
